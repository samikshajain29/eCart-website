# 🛒 eCart - MERN Stack E-commerce Website

**eCart** is a modern, full-stack e-commerce web application built with the **MERN stack** (MongoDB, Express.js, React.js, Node.js). It offers a complete online shopping experience with secure authentication, payment integration, and a feature-rich admin dashboard.

## 🚀 Features

- 🔐 Google OAuth Authentication  
- 💳 Razorpay Payment Gateway Integration  
- 🛍️ Product Browsing and Shopping Cart  
- 👤 User Registration & Profile Management  
- 📦 Order Placement & Tracking  
- 🛠️ Admin Panel  
  - Product Management  
  - Order Management  
  - User Management  
- 📱 Fully Responsive UI

## 🛠️ Tech Stack

- **Frontend:** React.js, React Router, Context API, CSS  
- **Backend:** Node.js, Express.js, MongoDB  
- **Authentication:** Google OAuth, JWT  
- **Payments:** Razorpay API  
- **Database:** MongoDB (Mongoose ODM)
